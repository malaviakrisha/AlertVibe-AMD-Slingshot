# AlertVibe: AMD Slingshot

**Non-Invasive Emergency Sound Recognition** for **Deaf & Elderly Safety**

## Problem

**Deaf and elderly** people miss **domestic dangers**:

- Fire alarms (30% miss critical alerts)
- Boiling kettles (burn risks)
- Glass shatters (injury hazards)

## Tech Stack

- **TensorFlow Lite** (95KB model)
- **MFCC + 3-Layer CNN** (90% accuracy target)
- **Flutter** (cross-platform)
- **AMD Ryzen AI** ready
